# السياسة الأمنية (Security Policy)

أهلاً بك! نحن نأخذ أمن "برق الذكي" على محمل الجد.

## الإصدارات المدعومة
حالياً، يتم دعم أحدث إصدار يتم رفعه على فرع `main` بالتحديثات الأمنية.

## الإبلاغ عن ثغرة أمنية
إذا اكتشفت ثغرة أمنية أو تسريب لمفاتيح (API Keys)، يرجى اتباع الخطوات التالية:

1. **عدم النشر العلني:** لا تقم بفتح "Issue" عام يتضمن تفاصيل الثغرة أو المفتاح المسرب.
2. **التواصل الخاص:** يرجى مراسلتي مباشرة عبر حسابي في GitHub أو عبر البريد الإلكتروني الخاص بالمطور (بارق).
3. **الاستجابة:** سأعمل على معالجة المشكلة وإبطال مفعول أي مفاتيح مسربة خلال 24 ساعة.

شكراً لمساعدتكم في الحفاظ على أمان "برق"!
